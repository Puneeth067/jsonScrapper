# JSON Data Scraper

## Overview

This script fetches employee data from an API, processes it, and stores it in CSV & Parquet.

## Setup

### Set Environment

```
python -m venv env
env/Scripts/activate
```

### Install Dependencies

```sh
pip install -r requirements.txt
```

### Run command

```
cd src
```

```sh
python main.py
```

### Test Command

```
cd src
```

```sh
python -m unittest tests.py
```
